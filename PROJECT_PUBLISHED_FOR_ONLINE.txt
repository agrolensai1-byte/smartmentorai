  2. Kill process: taskkill /PID [PID] /F
  3. Try again: node server.js

  1. Check remote: git remote -v
  2. Verify URL is correct
  3. Check credentials
  4. Try: git push origin main

  1. Check Render logs (Dashboard â†’ Logs)
  2. Verify Procfile exists
  3. Check package.json has npm start script
  4. Test locally first: npm install && node server.js
  5. Rebuild on Render dashboard

  1. Open Developer Tools (F12)
  2. Check Console tab for errors
  3. Check Network tab for failed requests
  4. Check Render logs for backend errors
